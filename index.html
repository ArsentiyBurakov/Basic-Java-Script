Basic Java-Script

Language basics Java-Script

1. Задать температуру в градусах по Цельсию. Вывести в alert соответствующую температуру в градусах по Фаренгейту.
Подсказка: расчёт идёт по формуле: Tf = (9 / 5) * Tc + 32, где Tf – температура по Фаренгейту, Tc – температура по Цельсию

var Tc = 30;
var Tf = (9 / 5) * Tc + 32;
alert(Tf);
Alert result: 86

2. Объявить две переменные: admin и name. Записать в name строку "Василий";
Скопировать значение из name в admin. Вывести admin (должно вывести «Василий»).

var name = "Василий";
var admin = name;
alert(admin);
Alert result: Василий

3. Чему будет равно JS-выражение 1000 + "108".

var a = 1000;
var b = "108";
alert(a + b);
Alert result: 1000108

5. Самостоятельно разобраться с атрибутами тега script (async и defer).

Атрибут async: загружает скрипт асинхронно, т.е. браузер не ждет выполнение скрипта.
Атрибут defer: откладывает выполнение скрипта до тех пор, пока вся страница не будет загружена полностью
